# Garfield Second Ward Emergency Preparedness

A static website providing emergency preparedness resources and information for members of the Garfield Second Ward of The Church of Jesus Christ of Latter-day Saints.

## Overview

This website provides guidance on:
- 72-Hour Emergency Kits
- Water Storage
- Food Storage
- Family Communication Plans

## Viewing the Website

This is a static HTML website that can be viewed by:
1. Opening `index.html` directly in a web browser
2. Hosting via GitHub Pages
3. Deploying to any static web hosting service

## GitHub Pages

To enable GitHub Pages:
1. Go to repository Settings
2. Navigate to Pages section
3. Select "Deploy from a branch"
4. Choose the main branch and root folder
5. Save to deploy
